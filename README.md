# ePortfolio

## Instalación dos módulos de Node js
```
npm install
```
Tamén tes que vincular o proxecto con Firebase cambiando a configuración en main.js.

### Montar un servidor de desenvolvemento
```
npm run serve
```

### Facer compilación de produción
```
npm run build
```
